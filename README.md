# create-typed-vite-react

CLI tool for creating TypeScript React projects with Vite from a template.

## Usage

With npm:

```bash
npx create-typed-vite-react
```

With yarn:

```bash
yarn create-typed-vite-react
```

Features

-   TypeScript setup
-   React configuration
-   Vite build tools
-   Customizable project settings
-   Git initialization
